# My Portfolio

Welcome to my portfolio!
This was created as a school project using Javascript and the Github API.
The goal was to demonstrate our understanding of JSON, working with APIs, and HTML/CSS/JavaScript.<br>

## Link to the deployed website 

[Github](https://clarahedgehog.github.io/portfolio/)

## Link to the GitHub repository 

[GitHub Repository](https://github.com/clarahedgehog/portfolio)

## Project structure 
- `index.html` - Home page <br>
- `about.html` - About Me page <br>
- `contact.html` - Contact Me page <br>
- `projects.html` - My Projects page <br>

- `about.js` - js file for the About Me page <br>
- `contact.js` - js file for the Contact Me page <br>
- `general.js` - js file for the general js <br>
- `projects.js` - js file for the My Projects page <br>

- `about.css` - file with styling for the About Me page.
- `contact.css` - files with styling for the Contact Me page.
- `general.css` - file with styling that is common across the whole website.
- `typography.css` - file for used fonts on the pages.
- `projects.css` - file with styling for the My Projects page.
- `index.css` - file with styling for the Home page.
